Rules

  • Blank line before and after:
    • Paragraphs
    • Lists
    • Code blocks
    • Blockquotes
  • indent list items with tabs to make nested lists
* list item 1
    * list item 1.1
    * list item 1.2
* list item 2

Nice to know

Markdown provides backslash escapes for the following characters:

\   backslash
`   backtick
*   asterisk
_   underscore
{}  curly braces
[]  square brackets
()  parentheses
#   hash mark
+   plus sign
-   minus sign (hyphen)
.   dot
!   exclamation mark
